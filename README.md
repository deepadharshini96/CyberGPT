# 🛡️ CyberGPT - AI-Powered Security Advisor

An intelligent chatbot that explains cybersecurity concepts using AI.

## ✨ Features

- 🤖 **AI-Powered Responses** - Uses OpenAI GPT to answer security questions
- 💻 **Code Examples** - Shows vulnerable and secure code snippets
- 📚 **Educational** - Learn about cybersecurity concepts
- 🎨 **Beautiful UI** - Modern, responsive interface
- 💾 **Chat History** - Keep track of your questions

## 🚀 Quick Start

### Prerequisites
- Node.js (v14+)
- OpenAI API key (get free credits at platform.openai.com)

### Installation

1. **Clone the repository*

2. **Setup Backend**
```bash
cd backend
npm install
echo OPENAI_API_KEY=your-key-here > .env
echo PORT=5000 >> .env
npm start
```

3. **Setup Frontend**
```bash
cd frontend
npm install
npm start
```

## 🔧 Technology Stack

- **Frontend:** React, Axios, React Markdown
- **Backend:** Express.js, OpenAI API
- **Deployment:** Vercel (frontend), Render (backend)

## 📝 Usage

1. Ask a security question
2. Get AI-generated responses with code examples
